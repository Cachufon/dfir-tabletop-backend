# DFIR Backend

This folder contains the shared technical backend that powers every DFIR service, keeping investigations, detections, and reporting aligned across teams. Each subdirectory below groups reusable artifacts so analysts can quickly find the material they need during engagements.

- **/playbooks**: step-by-step workflows for each service.
- **/rules**: detection and hunting rules.
- **/data_sources**: documentation for log and artifact sources.
- **/collectors**: scripts and tools to collect data from customer environments.
- **/triage**: host triage scripts (macOS and Windows) and profiles.
- **/normalization**: scripts and schemas for normalizing raw logs.
- **/report_templates**: standard report and executive summary templates.
