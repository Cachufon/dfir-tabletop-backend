# INTERNAL - macOS Finding Categories (Defensible Framework)

- Unauthorized Access
- Insider Misuse / Policy Violation
- Malware or Suspicious Software
- Persistence Mechanisms
- Data Access / Exfiltration
- Security Control Evasion
- System Integrity Issues
- No Evidence of Malicious Activity

Use neutral, defensible language and clearly document when no suspicious or malicious activity is identified to support HR/Legal review and closed-loop reporting.
