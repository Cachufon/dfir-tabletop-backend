# AI Scope Area Data Sources

This document describes the minimum and optional data sources required to run the AI scope area effectively.

## Minimum Data Sources (for AI scope area where applicable)

- Prompt logs or transcripts (if available)
- System/developer prompt artifacts (if available)
- Agent manifests / tool schemas (if applicable)

## Optional Data Sources

- Model gateway logs, policy logs, allow/deny logs
- Red-team / eval artifacts
- Access control and role mappings for AI systems
