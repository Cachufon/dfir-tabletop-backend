# AI Scope Area (DFIR Backend)

This is the AI scope area for Gruve Compromise Assessments. AI Safety / Prompt Security Assessment is Model 5 and leverages Gruve MAPS. The AI scope area may be referenced in Models 1–4 when AI telemetry exists, but Model 5 is the dedicated prompt and security evaluation model.

Artifacts under this scope area include:
- Data source documentation for AI-related telemetry and artifacts
- Guidance for exporting and collecting prompt and transcript data
- Detection logic and rules for AI-focused anomalies or misuse patterns
- Workflow guidance for running the AI scope area independently (Model 5) or alongside other scope areas when AI signals are present
