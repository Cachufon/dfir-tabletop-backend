# Identity Scope Area (DFIR Backend)

This folder contains backend assets for the Identity scope area of Gruve Compromise Assessments. The Identity scope area analyzes identity provider and authentication data (Okta, Entra / Azure AD, Google Workspace, etc.) to detect account takeover, misuse of access, risky sign-ins, MFA abuse, and suspicious admin activity.

Artifacts under this scope area include:
- Data source documentation for identity logs
- Collection and export guidance for IDPs
- Detection logic and rules for identity-related anomalies
- Workflow guidance for running the Identity scope area independently or as part of a broader Compromise Assessment
