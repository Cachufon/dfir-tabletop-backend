# Endpoint Scope Area (DFIR Backend)

This folder contains backend assets for the Endpoint scope area of Gruve Compromise Assessments. The Endpoint scope area analyzes endpoint telemetry from EDR products and host-level triage collections to detect persistence, suspicious processes, lateral movement, and attempts to evade controls.

Artifacts under this scope area include:
- Data source documentation for EDR and host artifacts
- Guidance for EDR exports and host triage collection
- Detection logic and rules for endpoint-focused anomalies
- Workflow guidance for running the Endpoint scope area independently or alongside Identity and SaaS scope areas
