# SaaS Scope Area (DFIR Backend)

This folder contains backend assets for the SaaS scope area of Gruve Compromise Assessments. The SaaS scope area analyzes business application activity (such as O365, Google Workspace, Slack, GitHub, and similar platforms) to detect mailbox manipulation, file exfiltration, abnormal sharing, risky OAuth usage, and suspicious collaboration behavior.

Artifacts under this scope area include:
- Data source documentation for SaaS platforms
- Guidance for exporting and collecting SaaS activity logs
- Detection logic and rules for SaaS-focused anomalies
- Workflow guidance for running the SaaS scope area independently or alongside Identity and Endpoint scope areas
