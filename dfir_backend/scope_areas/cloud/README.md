# Cloud Scope Area (DFIR Backend)

This is the Cloud scope area for Gruve Compromise Assessments. It focuses on AWS, GCP, and Azure telemetry to uncover IAM drift, access anomalies, cloud audit trail signals, misconfigurations, and cross-domain correlations with identity, SaaS, and endpoint activity.

Artifacts under this scope area include:
- Data source documentation for cloud providers
- Guidance for exporting and collecting cloud audit and configuration data
- Detection logic and rules for cloud-focused anomalies
- Workflow guidance for running the Cloud scope area independently or alongside other scope areas
