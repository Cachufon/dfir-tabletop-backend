# TTX Synthetic Demo Examples

These examples are synthetic demo cases generated by:
- `dfir_backend/ttx/scripts/seed_example_cases_and_run_end_to_end.py`

The generated cases are intended for deterministic, end-to-end workflow demonstration only.

## Output location

By default, outputs are written outside git under:
- `~/ttx_cases/examples`

Generated case folders:
- `001_saas_identity_compromise`
- `002_ransomware_extortion`
- `003_third_party_supply_chain`

## How to run

From repo root:
- `python3 dfir_backend/ttx/scripts/seed_example_cases_and_run_end_to_end.py`

Optional flags:
- `--out-root <path>` to choose a different output root
- `--timezone <tz>` to set case manifest timezone (default `America/Los_Angeles`)
- `--force` to overwrite existing example case folders

## Success criteria

For each generated case, success means all of the following exist:
- `20_delivery/ttx_package/` (built package)
- `20_delivery/handouts_html/` and `20_delivery/handouts_html/handouts_html.zip` (handouts export)
- `30_outputs/ttx_logs/scribe_runtime.json` (synthetic runtime logs)
- `30_outputs/after_action_report_draft.md` (AAR draft)
