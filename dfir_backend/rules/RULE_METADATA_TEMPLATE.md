## Name
Short, human-readable name.

## Format
One of: Sigma / YARA / IOC / AI_Prompt.

## Location
Path to the actual rule file (e.g. ./sigma/okta/impossible_travel.yml).

## Category
Identity / SaaS / Endpoint / Executive / Custom Threat Hunt.

## Data Sources

## Purpose
Short description of what this rule detects or validates.

## ATT&CK Mapping

## Severity Guidance
