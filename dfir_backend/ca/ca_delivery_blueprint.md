# CA Delivery Blueprint

## Ingestion Strategies

- export
- hunt-in-place
- remote collection

## Required Artifacts

Every engagement must produce, track, and hand off the following required artifacts:

- engagement scope record
- ingestion strategy record
- telemetry validation record
- normalization mapping record
- detection output record
- triage decision log
- findings register
- reporting package
- engagement limitations register

## Tooling Tiers

### v0 no-lab

Use when no dedicated lab infrastructure is available.

### v1 open-source

Use open-source tooling for ingestion, normalization, detection, triage, and reporting workflows.

### v2 commercial acceleration

Use commercial tooling to accelerate ingestion, analysis, triage, and reporting timelines.

## Limitations Documentation (Mandatory Every Engagement)

The following limitation categories must be documented for every engagement:

- data availability limitations
- data quality limitations
- timing and retention limitations
- access and permission limitations
- tooling coverage limitations
- detection fidelity limitations
- triage confidence limitations
- reporting completeness limitations

## Data Handling Warning

**DO NOT COMMIT CLIENT DATA**
