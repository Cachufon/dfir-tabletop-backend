# CA Delivery Documentation

This folder aligns Cyber Advisory (CA) delivery artifacts to the existing CA documentation set:

- `dfir_backend/workflows/ca_workflow.md`
- `dfir_backend/workflows/ca_models.md`
- `dfir_backend/runbooks/ca_engagement_runbook.md`

Use these files together:

- `ca_delivery_blueprint.md` defines ingestion strategies, required artifacts, tooling tiers, and engagement limitations.
- `case_workflow_state_machine.md` defines the CA case state machine and gate convergence across ingestion strategies.
